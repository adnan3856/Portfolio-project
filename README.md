Portfolio Backend V2 is a robust, production-grade API service designed to power a modern professional portfolio. Built with Java 21 and Spring Boot 3.4.2, this project represents a complete modernization of the backend infrastructure, focusing on scalability, security, and maintainability.

The service acts as a centralized data hub for managing professional identity, including detailed biographies, technical skill sets, work history, and academic achievements.

Key Features:
Modern Tech Stack: Leverages the latest features of Java 21 (LTS) and Spring Boot 3.x for optimal performance.
Data Persistence: Utilizes Spring Data JPA with a PostgreSQL database for reliable and structured data management.
Environment-Ready Configuration: Features a flexible configuration system using environment variables, making it ready for seamless deployment via Docker or cloud providers (AWS/Azure/GCP).
Clean Architecture: Follows a layered architecture (Controller, Service, Repository) to ensure clear separation of concerns and ease of testing.
Developer Efficiency: Integrated with Lombok to reduce boilerplate code and Spring Boot Actuator for real-time application monitoring and health checks.
Core Objective:
The primary goal of this project is to provide a high-performance, secure, and easily extensible RESTful API that can serve various frontend consumers (React, Angular, or Mobile) with professional profile data, replacing legacy monolithic structures with a microservice-ready design.
